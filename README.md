蓝湖css to windicss
</p>

这是一个基于 chrome plugin工具为了快速可以一键转换蓝湖的css代码成 windicss 原子类 inclass 代码

## 📦 use

```
 1. git clone 项目
```

```
 2. npm install
```

```
 3. npm run build 
```

```
 4. 浏览器-扩展程序-开发者模式-加载已解压的项目
```
